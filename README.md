# CONSUMER-CLIENT

Cliente que consume las API's Construdata, SIAB, Terceros

## Instalacion

composer require consultores/consumer-client


## Metodos

```
#Construdata

*listarCategorias
*crearCotizacion
*editarCotizacion
*agregarCantidad
*consultarCotizacion

#SIAB

*crearUserCiencuadras
*estadoCaso

#Terceros

*gestionTerceros

```
